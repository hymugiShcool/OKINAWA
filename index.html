沖繩 5 天 4 夜行程書（執行版）

住宿：Nest 那霸提薩吉飯店（Tissage Hotel Naha by Nest） — 沖繩縣那霸市西 2-14-1
官方：ティサージホテル那覇

飯店步行約 9～10 分至ゆいレール 「旭橋」站。那霸機場線：空港 → 県庁前官方標示約 13 分（全線與路線圖見 ゆいレール）。

DAY 1 那霸機場・飯店・國際通

行程動線

那霸機場 → Nest 那霸提薩吉飯店 → 國際通

交通

路段 方式
機場 → 飯店 ゆいレール：那霸機場站 → 旭橋（飯店步行至旭橋站）
飯店 → 國際通 ゆいレール：旭橋 → 縣廳前（國際通北側）；若要逛南段可改 牧志／美榮橋
自駕與單軌並用 車若停牧志：晚餐後可搭單軌 縣廳前／牧志 → 旭橋 回飯店，避免酒後駕車

若當日已租車：機場取車 → 飯店放行李 → 再開往牧志周邊停車場。

晚餐

選項 1（已預約 19:30）

  • 店家：島野菜とアグー豚・山城牛の店 こばこ

  • 店家：暖暮拉麵 那霸牧志店
  • 交通：🚐／自駕皆可；暖暮常排隊，建議預留 30～60 分

停車場（牧志／國際通周邊參考）

【 アップルパーク牧志2丁目第3 】

  • 支付：現金
  • 費率：60 分 ¥200（台幣約 40 元級，匯率依當日為準）

  • 地址：沖繩縣那霸市牧志 2-25-7

DAY 2 豬肉蛋飯糰・波上宮・第一牧志公設市場・PARCO CITY・燒肉王

行程動線

豬肉蛋飯糰 → 波上宮 → 第一牧志公設市場 → PARCO CITY → 燒肉王

交通

自駕（主軸）

  1. 飯店 → 豬肉蛋飯糰（牧志一帶）：約 5～10 分；停車用 Day 1 牧志停車場擇一
  2. → 波上宮：約 5～10 分；周邊搜「波の上ビーチ 駐車場」等公有／投幣場
  3. → 第一牧志公設市場：回牧志步行或短駕，停車同區
  4. → PARCO CITY（浦添）：一般道或自動車道約 25～45 分（路況浮動）
  5. → 燒肉王：以預約分店地址導航；自 PARCO 至那霸／浦添店多為 15～40 分

無車時

ゆいレール 牧志 出站 → 豬肉蛋與市場；波上宮以步行或計程車；PARCO 見下文「PARCO CITY 參考」；燒肉王以巴士或計程車接續。

餐食

餐別 內容
早餐 飯店早餐（請向飯店確認是否含早）＋豬肉蛋飯糰；若不含早可將飯糰當早午餐
午餐 第一牧志公設市場
晚餐 燒肉王（已預約 18:20） — 建議 15:30 前離開 PARCO，預留塞車與停車

PARCO CITY 參考

  • 地址：〒901-2123 沖繩縣浦添市西洲 3-1-1
  • 官網：parco city｜交通・停車：access（頁面載駐車場約 4,000 台免費等資訊，以現場為準）
  • 無車時：那霸巴士總站路線（例 385 等）或 おもろまち 周邊接駁／計程車；詳見官網與 沖繩巴士

DAY 3 古宇利大橋南端觀景台・蝦蝦飯・美麗海水族館

行程動線

古宇利大橋觀景台（南端）→ 蝦蝦飯 → 美麗海水族館

交通

  • 飯店 → 古宇利島：沖繩自動車道北上，單程多為 約 1.5～2.5 小時（含休息）；建議 07:00～08:00 前出發
  • 古宇利 → 蝦蝦飯：約 5～20 分（依店家位置）
  • 蝦蝦飯 → 海洋博公園／美麗海水族館：約 20～40 分
  • 回程那霸：至少預留 約 2 小時車程＋休息

餐食

餐別 內容
早餐 飯店早餐
午餐 蝦蝦飯（營業 11:00～16:00，務必在此時段內抵達）
晚餐 擇一：名護市區／許田道之驛／高速公路 SA 簡餐／回那霸再吃（最晚營業請事先查）

時間軸提醒

水族館認真逛＋海豚秀常需 3～4 小時；最晚入場與門票見 國營沖繩紀念公園／美ら海水族館。

停車場

地點 說明
古宇利大橋停車場 南端展望台周邊觀光客向駐車場；假日易滿，早到為宜（地圖搜「古宇利大橋 南側 駐車場」）
蝦蝦飯停車場／第二停車場 依現場看板與店員指示
美麗海水族館 海洋博公園內有料停車場（費率與平面圖以 官方 為準）

單日駕車時間長，建議兩人以上輪流開車，並備水與輕食。

行程動線

西來院（達摩寺）→ 泊港魚市場 → AEON MALL 來客夢 → 美國村

交通（車程為約略值）

路段 約略車時

泊港建議 09:00～10:30 前抵達較熱鬧。

餐食

午餐 泊港魚市場（丼、立食、熟食）
晚餐 AEON 來客夢 美食街或超市 ＋ 美國村 小食（塔可、海景酒吧、迴轉壽司等）；或擇一區吃正餐即可

停車場

DAY 5 國際通・免稅採購・Ashibinna・返台

行程動線（依航班調整）

  • 晚班機：退房 → 國際通 → 市區免稅（例：DFS／T ギャラリア沖縄，若你指的 DFF 為此類請核對店名） → Ashibinna OUTLET（豐見城、近機場）→ 還車 → 那霸空港
  • 早班機：退房 → Ashibinna 短停或略過 → 還車 → 報到；國際通採買建議改前一日

若為市區免稅店採購，請確認機場取貨櫃台與航班時間是否來得及。

交通

那霸市區 ↔ Ashibinna 自駕約 15～25 分；還車後依租車公司指示前往航廈。

餐食

餐別 內容
早餐 飯店早餐或國際通早午餐
午餐 國際通、牧志周邊，或 Ashibinna 內美食街
晚餐 早班或傍晚機：空港內簡餐；晚班機：市區吃完再進場

停車場

  • 國際通：可沿用 Day 1 牧志／県民広場 等投幣或地下場
  • DFS 等市區免稅：鄰近商業大樓付費停車場
  • Ashibinna：公式アクセス（多為免費大型停車場）

返台前緩衝

國際線建議預留 約 2.5～3 小時（含還車、報到、安檢、免稅取貨）；廉航依航空公司建議時間。
